Step-by-step Guide

01. Download and Install Node.js package
(https://nodejs.org/en/download)

02. Down and install VS Code
(https://code.visualstudio.com/)

03. Replace the deployment package for the desired the file directory

04. Open folder via VS Code

05. Open a new terminal and run the following commands to install the dependencies elated to the Node.js and Express framework

-> npm init -y
-> npm i express

06. Run the following command in the Terminal

-> node app.js

07. Go to the web browser and enter the following URL

-> localhost:8080 (to get the server up and runnning)
-> localhost:8080/operator/num1/num2 -> Ex: [localhost:8080/addition/3/2]

Enjoy! Cheers!
